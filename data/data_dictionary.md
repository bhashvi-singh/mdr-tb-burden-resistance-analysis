# Data Dictionary (Key Variables)

| Variable name        | Description |
|----------------------|-------------|
| country              | Country name |
| iso2                 | ISO 2-letter country code |
| iso3                 | ISO 3-letter country code |
| iso_numeric          | ISO numeric country code |
| g_whoregion          | WHO geographical region |
| year                 | Year of estimate |
| e_rr_pct_new         | Estimated % of rifampicin-resistant TB among new cases |
| e_rr_pct_ret         | Estimated % of rifampicin-resistant TB among previously treated cases |
| e_inc_rr_num         | Estimated number of MDR/RR-TB incident cases |
| e_inc_rr_num_lo      | Lower uncertainty bound for MDR/RR-TB incidence |
| e_inc_rr_num_hi      | Upper uncertainty bound for MDR/RR-TB incidence |

Descriptions follow WHO Global Tuberculosis Programme metadata.
